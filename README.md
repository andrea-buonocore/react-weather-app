
# Weather App

Dovrai creare una nuova applicazione per controllare il meteo in giro per il mondo!

Realizzata con React, Bootstrap, Redux.

```bash

```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

